# 🎬 AI Video Summarizer

🚀 Turn long videos into short, meaningful summaries using AI

---

## ✨ Features

* 🎥 Upload any video
* 🔊 Automatic audio extraction
* 🗣️ Speech-to-text using faster-whisper
* 🧠 AI-powered summarization (LLM via Ollama)
* ⚡ Fast + accurate results
* 🎉 Fun interactive UI experience

---

## 🎯 Demo Flow

```text
Upload Video → Processing → Transcript → AI Summary
```

---

## 🖥️ Tech Stack

* 🐍 Python
* 🎬 MoviePy
* 🧠 faster-whisper
* 🤖 Ollama (Mistral / LLaMA3)
* 🌐 Streamlit

---

## 🚀 How to Run Locally

### 1️⃣ Clone repo

```bash
git clone https://github.com/your-username/ai-video-summarizer.git
cd ai-video-summarizer
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Install Ollama & model

```bash
ollama run mistral
```

### 4️⃣ Run app

```bash
streamlit run app_ui.py
```
